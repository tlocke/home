Non-tracking blog

Sun 06 May 2018

My website and blog were hosted by Google, which tracks people on an industrial scale. So I've now moved to GitHub Pages using a site that's statically generated using Pelican. GitHub Pages have better privacy than Google, but the deeper point is that a static site stored in a git repository can be easily moved to another host.

So, this is the first post on the new blog!
