Web Apps

Published: Sun 02 March 2008
By Tony Locke

In misc.

This is how I'd like various web apps to work.

Common

  • Should support OpenId.

Wiki

  • The URLs to the pages would be constructed in accordance with DesigningURLs. Each page would be assigned a serial number. This number would never change. There would be no renaming of URLs.
  • It should be possible to edit the raw HTML.

Bookmarking

  • When a new URL is added, the default title should be the title found in the markup (<title> element in HTML)
  • Clicking on a link should open it in a new tab

links

  • Pelican
  • Python.org
  • Jinja2
  • You can modify those links in your config file
