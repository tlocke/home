Own-Id: DNS propogation bug

Thu 08 November 2007

I was going through the Own-Id error log and found a bug that affects new users only. It occurred when the DNS propagated to the user's browser before it reached the Own-Id server. I've now changed the code so that no DNS lookup is ever done on the domain name that the user is trying to set up.
